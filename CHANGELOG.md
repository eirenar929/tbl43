# Changelog

Все заметные изменения в проекте "Столик на троих" будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
а проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [2.0.0] - 2024-05-09

### Добавлено
- Единый экземпляр Playwright для всех WebChatAdapter (оптимизация ресурсов)
- Пул HTTP-клиентов для переиспользования соединений
- JSON-логирование диалогов в `logs/dialogs.jsonl`
- Graceful shutdown для корректного закрытия компонентов
- Pydantic Settings для валидации конфигурации через переменные окружения
- Расширенный health-check со статистикой по каждой модели
- Обновлённый UI с тёмной темой
- Улучшенная обработка ошибок
- Оптимизированный Dockerfile

### Изменено
- Улучшена архитектура буфера ответов с логированием
- Оптимизирована работа с WebSocket соединениями
- Улучшена документация проекта

### Исправлено
- Race condition при установке callback в ResponseBuffer
- Проблемы с утечкой памяти при долгой работе сервера
- Некорректная обработка таймаутов моделей

## [1.0.0] - 2024-01-01

### Добавлено
- Первоначальная версия проекта
- Базовая поддержка нескольких ИИ-моделей
- Простая модерация контента
- WebSocket интерфейс для чата
